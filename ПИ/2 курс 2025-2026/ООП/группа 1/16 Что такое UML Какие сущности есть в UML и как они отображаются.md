- **Что такое?** UML — это стандартизированный **язык графического моделирования** для визуализации, спецификации, проектирования и документирования архитектуры программных систем.
    
- **Сущности и их отображение:**
    
    - **Класс:** Прямоугольник, разделенный на 3 секции: **Имя класса | Поля (атрибуты) | Методы (операции)**. Модификаторы доступа: `+` (public), `-` (private), `#` (protected).
```text
┌─────────────────────┐
│      BankAccount    │
├─────────────────────┤
│ - balance: double   │
│ - owner: string     │
├─────────────────────┤
│ + deposit(amount)   │
│ + withdraw(amount)  │
│ + getBalance(): double│
└─────────────────────┘
``` 

- **Объект:** Прямоугольник с подчеркнутым именем в формате `имя_объекта : Имя_Класса`.
    - **Отношения:**
        - **Наследование:** Сплошная линия с **пустой треугольной стрелкой** от наследника к родителю.
        - **Ассоциация (связь):** Сплошная линия (может быть со стрелкой и кратностью, напр., `1`, `*`, `0..1`).
        - **Агрегация (часть-целое, "has-a"):** Сплошная линия с **пустым ромбом** со стороны целого.
        - **Композиция (сильная агрегация, "состоит из"):** Сплошная линия  **заполненным ромбом** со стороны целого.
