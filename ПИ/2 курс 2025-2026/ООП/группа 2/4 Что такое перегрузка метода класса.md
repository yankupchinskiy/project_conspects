**Перегрузка метода** — это определение нескольких методов с одним именем, но разными параметрами (по типу, количеству или порядку). Компилятор выбирает подходящий метод на основе переданных аргументов.
``` c++
class Calculator {
public:
    int add(int a, int b) { return a + b; }
    double add(double a, double b) { return a + b; }
    int add(int a, int b, int c) { return a + b + c; }
};
```
