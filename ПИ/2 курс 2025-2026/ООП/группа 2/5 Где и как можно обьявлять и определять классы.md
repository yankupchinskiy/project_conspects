**Объявление класса** обычно размещается в заголовочных файлах (`.h`, `.hpp`):

cpp

``` c++
// MyClass.h
#pragma once
class MyClass {
public:
    void method();
};
```
**Определение методов** может быть:

- Внутри класса (автоматически inline):
```c++
class MyClass {
public:
	void method() { /* реализация */ }
};
```
- Вне класса, в файле реализации (`.cpp`):
``` cpp
   // MyClass.cpp
  #include "MyClass.h"
   void MyClass::method() { /* реализация */ }``
```
