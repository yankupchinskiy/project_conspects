Компилятор C++ автоматически генерирует (если программист не определил их явно):

1. Конструктор по умолчанию
    
2. Конструктор копирования
    
3. Оператор присваивания копированием
    
4. Деструктор
    
5. Конструктор перемещения (C++11)
    
6. Оператор присваивания перемещением (C++11)
    

**Явное указание для генерации по умолчанию:**
``` c++
class MyClass {
public:
    MyClass() = default; // Генерация конструктора по умолчанию
    MyClass(const MyClass&) = default; // Генерация конструктора копирования
    ~MyClass() = default; // Генерация деструктора
};
```
