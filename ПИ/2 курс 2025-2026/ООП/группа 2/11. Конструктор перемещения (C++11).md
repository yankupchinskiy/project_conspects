
**Конструктор перемещения** "перемещает" ресурсы из временного объекта:
```c++
class MyClass {
public:
    MyClass(MyClass&& other) noexcept {
        // Перемещение ресурсов из other
        // other остается в допустимом, но неопределенном состоянии
    }
};
```

**Неявный вызов происходит** при работе с rvalue-выражениями:

``` c++
MyClass createObject() { return MyClass(); }
MyClass obj = createObject(); // Может вызвать конструктор перемещения
```
