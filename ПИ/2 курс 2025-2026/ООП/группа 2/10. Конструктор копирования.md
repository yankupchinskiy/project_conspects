**Конструктор копирования (copy constructor)** — это специальный конструктор, который создаёт новый объект как копию существующего объекта того же класса. Он гарантирует, что при копировании объекта все его члены будут правильно скопированы.
``` c++
class MyClass {
public:
    MyClass(const MyClass& other) {
        // Копирование данных из other
    }
};
```
**Неявный вызов происходит:**

- При передаче объекта в функцию по значению
    
- При возврате объекта из функции по значению
    
- При инициализации: `MyClass obj2 = obj1;`
