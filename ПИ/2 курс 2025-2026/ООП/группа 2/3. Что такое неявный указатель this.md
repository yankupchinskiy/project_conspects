**`this`** — это неявный указатель, доступный внутри любого нестатического метода класса, который указывает на объект, для которого был вызван метод.
```c++
class MyClass {
    int x;
public:
    void setX(int x) {
        this->x = x; // this->x - поле класса, x - параметр
    }
    
    MyClass* getAddress() {
        return this; // Возвращает указатель на текущий объект
    }
};
```
