В контексте классов:

1. **Статические поля**: одна копия на весь класс
```c++
    class Counter {
        static int count; // Общее для всех объектов
    };
```
2. **Статические методы**: не имеют доступа к `this`
  ```c++
    class Math {
    public:
        static double sqrt(double x);
    };
  ```
